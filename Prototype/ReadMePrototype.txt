# README Prototipo – Study Room
## Descripción
Este prototipo es una **prueba de concepto (PoC)** del proyecto *Study Room*, una app de salas de estudio virtuales con estilo pixel art.
---
## Objetivo
Validar la experiencia básica:
* Ingresar un nombre
* Entrar a una sala
* Ver un personaje
* Moverse dentro del espacio
---
## Alcance
Solo frontend. No incluye:
* Backend
* Login real
* Base de datos
---
## Funcionalidades
* Pantalla inicial (input de nombre)
* Navegación a la sala
* Sala de estudio
* Personaje con nombre
* Movimiento básico
* Elementos visuales (pomodoro, música, to-do, etc.)
---
## Tecnologías
* React (Vite)
* JavaScript
* CSS
---
##  Trabajo en equipo
Cada integrante trabaja una parte (en ramas separadas) y luego se integra todo:
* Login
* Sala
* Personaje
* Movimiento
* Interacciones
---
