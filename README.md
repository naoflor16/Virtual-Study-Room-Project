# Virtual-Study-Room-Project
Proyecto para crear una sala virtual interactiva donde los usuarios
puedan estudiar, escuchar música y gestionar su tiempo mediante
herramientas como pomodoro.

Concepto:
Un espacio visual tipo habitación donde el usuario controla un
personaje que puede desplazarse hacia diferentes zonas funcionales
(música, temporizador, etc.).

Objetivo del PoC:
Demostrar la interacción básica dentro del entorno virtual y la
integración inicial de las herramientas de estudio.
